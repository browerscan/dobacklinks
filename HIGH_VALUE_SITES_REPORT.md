## 📊 流量分布统计

### 按流量级别分类

```sql
-- 查询不同流量级别的站点数量
SELECT
  CASE
    WHEN monthly_visits >= 10000000 THEN '10M+ (超高流量)'
    WHEN monthly_visits >= 1000000 THEN '1M-10M (高流量)'
    WHEN monthly_visits >= 100000 THEN '100K-1M (中高流量)'
    WHEN monthly_visits >= 10000 THEN '10K-100K (中等流量)'
    ELSE '<10K (低流量)'
  END as traffic_tier,
  COUNT(*) as site_count,
  AVG(monthly_visits) as avg_visits
FROM products
WHERE enrichment_status = 'enriched'
GROUP BY traffic_tier
ORDER BY AVG(monthly_visits) DESC;
```

**预期分布**:

- **10M+ 超高流量**: ~15 个站点
- **1M-10M 高流量**: ~30-40 个站点
- **100K-1M 中高流量**: ~80-100 个站点
- **10K-100K 中等流量**: ~100-120 个站点
- **<10K 低流量**: ~20-30 个站点
